[#appendix-assertion-template]
= 附录 B：断言模板

断言使用标题承载稳定引用地址。模板如下。

[source,asciidoc]
----
[#stable-assertion-id.assertion, normative=must]
=== 读者可见断言标题

前置状态:: 用户和系统在动作发生前可观察到的状态。

用户动作:: 用户执行的动作，或外部可观察事件。

期望结果:: 系统应出现的用户可观察结果。

不符合表现:: 会导致断言失败的典型行为。

覆盖对象:: xref:camera-permission[摄像头授权, rel=verifies]。
----

断言标题应表达行为判定，而不是内部实现。`normative` 字段使用 `must`、`should` 或 `may`。断言正文检查用户表面，不检查内部代码。
