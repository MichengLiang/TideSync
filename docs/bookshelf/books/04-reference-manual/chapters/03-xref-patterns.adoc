[#xref-patterns]
== 引用模式

书内引用使用本书内 anchor：

[source,asciidoc]
----
xref:#decision-table[结构选择表]
----

跨书引用写向 AsciiDoc 源文件：

[source,asciidoc]
----
xref + : + ../other-book/book.adoc#stable-anchor[目标章节]
----

上面的写法用分段文本展示，避免默认样本之间形成真实链接依赖。实际写作时去掉空格和加号，写成一个完整的 cross reference 宏。

被跨书引用的位置应写显式 anchor。显式 anchor 是维护者可以稳定依赖的目标。
