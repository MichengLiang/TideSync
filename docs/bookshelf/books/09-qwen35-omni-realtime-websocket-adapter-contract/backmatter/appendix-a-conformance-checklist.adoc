= 附录 A：符合性检查表

本检查表供 PR 作者、评审者和 QA 使用。

== 源码与运行路径

* [ ] Qwen adapter 派生源码进入 TideSync 仓库。
* [ ] PR diff 可见主要源码修改。
* [ ] 受控源码目录包含上游来源说明。
* [ ] 依赖解析指向受控源码。

* [ ] 客户端事件集合完整实现。
* [ ] 服务端事件集合完整处理。

* [ ] response state 绑定 response id。
* [ ] audio done 形成 output done。
* [ ] speech_started 触发用户 turn、本地 flush、远端 cancel 和旧 response 隔离。
